EduWave Academy - Student Learning Dashboard

This is a complete student learning dashboard built with HTML, CSS, and JavaScript. It includes multiple pages for tracking courses, quizzes, progress, and managing reminders.

Installation Guide

  1. Download all the files and maintain the folder structure.
  2. Make sure you have an internet connection for loading the external libraries:
    • Font Awesome (for icons)
    • Chart.js (for charts)
    • Flatpickr (for date picker)
  3. Open index.html in your web browser to start using the dashboard.
Note: This is a frontend-only implementation. In a real-world application, you would connect this to a backend server for data persistence.

Features

  • Dashboard: Overview with stats cards and charts
  • Courses: List of courses with progress tracking
  • Quizzes: Upcoming and past quizzes
  • Progress: Detailed progress tracking with charts and tables
  • Reminders: Create, edit, and manage reminders
  • Settings: User profile and application settings
  • Responsive Design: Works on mobile and desktop devices
  • Dark Mode: Toggle between light and dark themes

Usage

  • Navigate between pages using the sidebar
  • Add reminders by clicking the "Add Reminder" button
  • Edit your profile in the settings page
  • Toggle between different views using the tabs
  • Track your course progress and upcoming quizzes
